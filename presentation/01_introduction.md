# 📑 [Slide 01] Introduction: Problem Definition & Goals

## 1. 발표 자료 개요 (Slide Content)

### 📌 연구 배경: 심볼릭 음악 생성 (Symbolic Music Generation)
* **심볼릭 음악 생성의 의의**: 웨이브폼(Waveform)을 직접 생성하는 대신, 음악의 구조적 본질인 **MIDI 신호(Note, Velocity, Duration, Tempo)**를 생성하여 작곡의 논리성을 확보하고 실시간 인터랙션 성능을 달성하는 기계학습 분야.
* **목표**: 연주자(멜로디 입력)의 실시간 연주에 반응하여 풍부하고 하모닉한 피아노 반주(Accompaniment)를 즉흥적으로 생성하는 **Melody-Conditioned Jam Transformer** 구축.

### ⚠️ 기존 베이스라인 모델들의 2대 한계점
1. **절대 음높이(Absolute Pitch) 표현의 Sparsity 문제**
   * 기존 REMI/Music Transformer는 절대 MIDI 번호(0~127)를 그대로 학습함.
   * C Major 조성에서 학습한 화성 진행 패턴이 F# Major로 조옮김되면 완전히 새로운 토큰 조합으로 간주되어 전조 일반화 성능이 낮고 막대한 전조 증강 데이터와 대형 매개변수 모델이 요구됨.
2. **단선율 반주로의 모드 붕괴 (Single-Note Accompaniment Collapse)**
   * 크로스 엔트로피 손실 함수 특성상, 학습이 진행될수록 모델이 복잡한 화음(Chord)을 쌓기보다 손실이 낮고 안전한 단선율 반주(Monophonic texture)로 정렬 및 수렴하는 경향을 보임.
   * 이로 인해 AI가 생성한 반주가 음악적으로 메마르고 단순한 양손 단선율에 고착화되는 문제가 발생.
